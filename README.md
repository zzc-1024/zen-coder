# 项目介绍

拖拖拽拽，生成代码。项目灵感来自于虚幻引擎的蓝图系统，用户可以通过拖拽组件来创建代码。通过分层表示，可以将统一的蓝图翻译为不同的代码语言，目前只实现了 `Python` 语言。

# 快速开始

首先克隆项目，建议使用 `nvm` 来管理 `node` 版本。

```bash
# 使用的 node 版本为 22.x
nvm install 22
# 切换到 22.x 版本
nvm use 22
```

解决 `node` 版本问题后，开始启动项目。

首先切换到项目工作目录：

```bash
cd apps/web
```

安装依赖：

```bash
pnpm install
```

启动项目：

```bash
pnpm dev
```
